# 🚀 BookEase – Smart Online Appointment Booking System

A modern full-stack appointment booking platform built with React, Node.js, Express.js, and Supabase, enabling users to book appointments online with dynamic time-slot management and an admin dashboard.

## Folder Structure
```text
bookease/
├── backend/
├── docs/
├── frontend/
└── README.md
```

## Getting Started
To get started:
1. Clone the repository.
2. Navigate to the backend directory (`cd backend`), install dependencies, and run with `npm run dev`.
3. Navigate to the frontend directory (`cd frontend`), install dependencies, and run with `npm run dev`.
